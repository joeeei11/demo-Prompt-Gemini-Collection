# demo-Prompt-Gemini-Collection

**Gemini、Gemini CLI 与 Nano 模型的常用提示词合集，按场景分类，持续更新。**

## 功能特性

- 覆盖 Gemini / Gemini CLI / Nano 系列模型
- 按场景整理：编程、写作、分析、创意等
- Markdown 格式，便于直接复制使用
- 持续收录社区优质提示词

## 快速开始

```bash
git clone https://github.com/joeeei11/demo-Prompt-Gemini-Collection.git
```

打开 README.md 找到所需场景，复制提示词直接使用。

### 基础用法

```
将提示词粘贴到 Gemini / Gemini CLI 中即可使用
```
